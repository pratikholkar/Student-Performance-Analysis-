# 🎓 College Student Performance Analytics Dashboard

> **An End-to-End Data Analytics Project using Python, SQL Server, and Power BI**

---

# 📌 Project Overview

Educational institutions generate large volumes of student-related data, including academic records, attendance, certifications, internships, projects, placement information, and career readiness assessments. However, this data is often scattered across different systems and remains underutilized for academic planning and decision-making.

This project demonstrates a complete **End-to-End Data Analytics workflow** by transforming raw educational ERP data into meaningful business insights using **Python, SQL Server, and Power BI**.

The solution enables educational institutions to:

- Monitor academic performance
- Analyze attendance trends
- Evaluate internship participation
- Measure student skill development
- Assess placement readiness
- Identify academically at-risk students
- Support data-driven academic decision-making

---

# 🎯 Business Problem

Educational institutions often maintain student data across multiple systems, making it difficult to analyze academic performance, student engagement, and placement readiness effectively.

This project addresses the following business question:

> **How can educational institutions leverage student data to identify performance trends, improve student engagement, enhance placement readiness, and support data-driven academic decision-making?**

# 📈 Key Insights

## Academic Insights

- Students with higher attendance generally achieve better academic performance.
- Academic performance varies across engineering departments.
- Attendance has a positive relationship with CGPA.

## Skills & Development Insights

- Students with more certifications and projects demonstrate higher placement readiness.
- Hackathon participation contributes positively to practical skill development.

## Placement Insights

- Internship experience significantly improves placement opportunities.
- Higher resume scores and aptitude performance increase placement success.

## Risk Insights

- Low attendance and active backlogs are strong indicators of academic risk.
- Early identification of at-risk students enables timely academic intervention.
